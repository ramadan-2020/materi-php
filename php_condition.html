Condition dalam PHP bisa menggunakan:
- if
- if .. else
- if .. else if ... else
- switch


IF
Digunakan untuk pilihan satu kondisi Contoh:
<?php

$harga = 3000;

if($harga < 5000){
echo "Saya beli";
}

?>

Hasilnya:



IF ... ELSE
Digunakan untuk dua pilihan, ya atau tidak. Contoh:
<?php

$harga = 3000;

if($harga < 5000){
echo "Saya beli";
}else{
echo "Tidak saya beli";
}

?>

Hasilnya:


Keterangan:
Cobalah mengubah-ubah nilai harga


IF ... ELSE IF ... ELSE
Digunakan untuk beberapa pilihan. Contoh:
<?php

$harga = 3000;

if($harga > 5000){
echo "Tidak saya beli";
}else if($harga > 2000){
echo "Saya tawar";
}else{
echo "Saya beli";
}

?>

Hasilnya:


Keterangan:
Cobalah mengubah-ubah nilai harga


SWITCH
Digunakan untuk beberapa pilihan. Contoh:
<?php

$harga = 3000;

switch($harga){
case ($harga > 5000):
echo "Tidak saya beli";
break;
case ($harga > 2000):
echo "Saya tawar";
break;
default:
echo "Saya beli";
}

?>

Hasilnya:


Keterangan:
Cobalah mengubah-ubah nilai harga
