Website ada 2 macam, statis (isi halamannya tetap) dan dinamis (isi halamannya bisa berubah-ubah). Untuk membuat halaman statis cukup dengan HTML dan CSS. Sedangkan untuk membuatnya dinamis, kita bisa menggunakan PHP dan biasanya dihubungkan dengan database.

Pada tutorial kali ini akan kita bahas mengenai bagaimana menghubungkan php dengan database MySQL.

Yang pertama kita butuhkan adalah webserver. Bagi yang smartphone-nya sudah dipasang webserver dan support mysql, silahkan bisa langsung dipraktikkan di smartphonenya. Bagi yang belum, bisa praktik di komputer dengan menginstal XAMPP.

Saya anggap masalah server sudah siap. Lanjut menghubungkan php dengan mysql. Caranya seperti berikut:

<?php

$hostname = "localhost";
$username = "root";
$password = "";

// buat connection
$conn = mysqli_connect($hostname, $username, $password);
// Cek connection
if ($conn) {
    echo "Connection seccess";
}else{
    die("Connection failed: " . mysqli_connect_error());
}



?>

Hasilnya:


Jika berhasil, sekarang kita lanjutkan membuat database. Perintah/query yang digunakan adalah CREATE DATABASE nama_database atau bisa ditambahkan IF NOT EXISTS artinya database akan dibuat jika belum pernah dibuat sebelumnya sehingga query-nya menjadi CREATE DATABASE IF NOT EXISTS nama_database. Untuk menjalankan query-nya dengan perintah mysqli_query($connection, "query"). Berikut kita praktikkan, tulis kode berikut di bawah kode connection.


// kode connection

// Create database
$sql = "CREATE DATABASE IF NOT EXISTS mydb";
if (mysqli_query($conn, $sql)) {
    echo "sukses create db";
} else {
    echo "error create db";
}


Hasilnya:


Setelah database berhasil dibuat, kita perlu memilih database tersebut untuk digunakan pada program selanjutnya. Caranya seperti berikut:


// kode connection
// kode create database

// select database
mysqli_select_db($conn, "mydb");
